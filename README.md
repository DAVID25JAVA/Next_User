Next.js User CRUD App
📌 Overview

Simple CRUD app using Next.js App Router and JSONPlaceholder API.
Users can view, update, and delete data.

🚀 Features
Fetch all users
View single user
Update user (with validation)
Delete user
Optimistic UI updates
Loading & toast messages


🛠 Tech Stack
Next.js
React
JavaScript
Axios
Tailwind CSS


⚡ How to Run
git clone <your-repo-link>
cd project-folder
npm install
npm run dev

Open: http://localhost:3000
